# EOAPP

Aplicación web sencilla para gestionar ingresos, gastos y metas personales.

Descripción
- Proyecto educativo y práctico que combina un backend en Node/Express y un frontend en React + Vite.

Referencia
- Repositorio relacionado: https://github.com/salaminamagdalena2014-hub/eoapp

Instalación rápida
- Backend:
	1. `cd eoapp-backend`
	2. `npm install`
	3. `npm start`
- Frontend:
	1. `cd eoapp-react`
	2. `npm install`
	3. `npm run dev`

Estructura principal
- `eoapp-backend`: servidor, rutas y base de datos.
- `eoapp-react`: interfaz de usuario con Vite y React.
